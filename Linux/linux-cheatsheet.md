# Linux Cheat Sheet
Personal notes written while learning Linux for DevOps.
Simple. Practical. No fluff.

---

## Navigation
```bash
pwd        # shows current directory
ls         # list files
ls -la     # list all files + permissions
cd folder  # move into folder
cd ..      # go back
```

## File Management
```bash
touch      # Create an empty file
cat        # View content in a file (Inside the terminal, text only)
rm         # Delete a file
rm -r      # Delete a directory

```
